# Пульт охраны банка

Это внутренний репозиторий для сотрудников банка «Сияние». Если вы попали в этот репозиторий случайно, то вы не сможете его запустить, т.к. у вас нет доступа к БД, но можете свободно использовать код вёрстки или посмотреть как реализованы запросы к БД.

Пульт охраны - это сайт, который можно подключить к удалённой базе данных с визитами и карточками пропуска сотрудников нашего банка.

## Запуск

Для запуска пульта охраны у вас уже должен быть установлен Python 3.

- Установите зависимости командой `pip install -r requirements.txt`
- Определите настройки доступа к базе данных. Для этого создайте файл `.env` рядом с `main.py` и запишите туда данные в таком формате: `ПЕРЕМЕННАЯ=значение`.  
    Требуются следущие переменные:
    - `DB_ENGINE` — движок базы данных.
    - `DB_HOST` — хост, на котором расположена база данных.
    - `DB_PORT` — номер порта для подключения к базе данных.
    - `DB_NAME` — имя базы данных.
    - `DB_USER` — имя пользователя для авторизации.
    - `DB_PASSWORD` — пароль пользователя для авторизации.
    - `SECRET_KEY` — секретный ключ приложения.
- Запустите приложение командой `python .\main.py`. Если настройки в файле `.env` введены верно и у вас есть доступ к базе данных, вы увидите количество карточек пропусков - всех и только активных.
